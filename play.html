Desktop Installation​

  1. Access the Online Patcher by clicking the Online Patcher button above
  2. Upload your ROM by clicking the "Rom File" button
  3. Click the "Patch" button
  4. Download the patched ROM file when complete

Congratulations! 🎉 You've patched your vanilla Pokémon Emerald rom into Emerald Imperium. Now, play it with your chosen emulator.

Mobile Installation​

Manual Patch​

  1. Download the Patch
    Click the "Download Patch" button above to get the .bps patch file.
  2. Get Your ROM
    Ensure you have a clean copy of Pokémon Emerald (USA) v1.0.
  3. Open a Patching Tool
    Use Rom Patcher JS (web-based) or any compatible patching tool.
  4. Apply the Patch
    • Select your ROM file as the "ROM file".
    • Select the downloaded .bps file as the "Patch file".
    • Click "Apply patch".
  5. Download the Patched ROM
    Save the output file (rename to .gba if needed).

You can now play Emerald Imperium on your favorite emulator!
